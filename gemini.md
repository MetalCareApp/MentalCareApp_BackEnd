## 2. 기술 스택 (Tech Stack)
- **언어 및 프레임워크**: Java 21 / Spring Boot 4.0.4
- **데이터베이스**: MySQL
- **API 테스트**: Postman

## 3. 추가 사항
- 안드로이드 앱 프로그램
- 백엔드 개발
- 현재 MySQL 연동이 되어있는 상태

## 4. 권한 및 가입 정책
- **기본 권한**: 최초 회원가입 시 모든 사용자는 기본적으로 '환자(Role.USER)' 권한을 가짐.
- **의사 권한 획득**: 가입 후 마이페이지 내 '의사 모드 전환' 버튼을 통해 별도의 인증/전환 절차를 거쳐 '의사(Role.DOCTOR)' 권한을 얻음.
- **연동 전적 기반 가입**: 
    - 의사-환자 연동 시스템을 이용한 기록이 있는 경우: 별도 복잡한 절차 없이 간편 가입 진행.
    - 기록이 없는 경우: 일반적인 회원가입 절차를 통해 계정 생성.
